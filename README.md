## Development

  - Dev: `npm run dev`
    - Work on `./dev/index.js`
    - CSS: `/dev/style.css`
  - Build: 
    - `npm run build`
    - `npm run server` (manual testing)
    - `git add; git commit -m "Build"; git push`
