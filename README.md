# JavaScript Visualized and Animated

[Demo (work in progress)](https://in-tech-gration.github.io/JavaScript.Animated/dev/index.html)

## Development

  - Dev: `npm run dev`
    - Work on `./dev/index.js`
    - CSS: `/dev/style.css`
  - Build: 
    - `npm run build`
    - Copy `dist/dev.js` and `dist/dev.map.js` to `dev/`
